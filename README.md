# Portfolio statique - scaffold

Ce dépôt contient un scaffold pour un portfolio statique : page d'accueil, lien pour télécharger un CV (`assets/cv.pdf`) et deux mini-jeux (`games/tictactoe.html`, `games/memory.html`).

Remarques rapides :
- Ajoutez votre PDF `assets/cv.pdf`.
- Pour déployer sur GitHub Pages : poussez sur `main` (ou `master`) et activez Pages depuis les settings ou utilisez le workflow fourni.

Pour servir localement :

Windows (PowerShell):
```
# simple serveur Python
python -m http.server 8000

# puis ouvrir http://localhost:8000
```
